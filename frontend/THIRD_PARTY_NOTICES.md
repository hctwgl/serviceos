# 第三方工程基线说明

ServiceOS Web Workspace 的工程组织参考 Vue Vben Admin 5.7.0 `web-antd`：

- 上游仓库：<https://github.com/vbenjs/vue-vben-admin>
- 许可证：MIT License
- 固定参考版本：5.7.0

ServiceOS 只保留 Vue 3、Vite、pnpm Workspace、Pinia、Vue Router、请求层和 Ant Design Vue
适配方向。上游 Demo、Mock、多 UI 库、多主题、暗色模式、示例业务和模板偏好中心均未引入；
本仓库不与上游业务页面同步，也不提供 Vben 兼容层。
