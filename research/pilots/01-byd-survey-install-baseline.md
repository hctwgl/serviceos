# 比亚迪勘安试点业务基线

## 1. 文档目的

本文件用于冻结 ServiceOS 首个端到端业务试点范围，避免在业务事实尚未稳定时同时开发全部车企、品牌、区域和业务类型。

本文件不是完整车企合同，也不替代正式项目规则。只有经业务负责人、产品负责人和技术负责人共同确认后，方可由 `Draft` 升级为 `Accepted` 并作为研发输入。

## 2. 试点范围

| 维度 | 基线 |
|---|---|
| 车企集团 | 比亚迪 |
| 品牌范围 | 首期选择一个品牌，默认从海洋或王朝中确定 |
| 业务类型 | 勘测后安装（勘安） |
| 区域范围 | 选择一个工单量适中、网点结构完整的省或城市群 |
| 工单来源 | 车企接口推送；接口未就绪时允许受控导入模拟 |
| 履约主体 | 总部、项目经理、客服、网点负责人、师傅 |
| 审核主体 | 总部客服审核，车企外部审核作为回传结果 |
| 结算 | 首
