# 页面清单与设计等级

状态：Accepted

## 设计等级

### A 级：完整高保真设计

必须具备完整页面、关键交互、异常状态、页面读模型和验收标准。

### B 级：标准页面模式加业务规格

复用已接受的页面模式，但必须明确字段、筛选、动作、权限和状态。

### C 级：规范覆盖

低频、只读或治理型页面，依据统一模式实现并完成基本验收。

## 通用管理页面

客户、用户、角色、网点、师傅、服务目录、消息和审计等页面，在核心页面模式确认后按 B 级或 C 级设计，不在第一批中逐页创造新视觉。

## 设计顺序

第一批按以下顺序推进：

1. 工单工作区；
2. 项目履约配置中心；
3. Admin 工作台；
4. Admin 工单中心；
5. 派单与改派；
6. 流程配置与版本发布。

先用最复杂的工作区和配置中心验证 A+ 是否能够承载真实业务，再固化其他页面。
