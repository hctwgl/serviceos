# ADR-002：工单锁定版本化配置包

- 状态：Proposed
- 日期：2026-07-12

## 背景

车企、品牌、年度、区域、桩型和业务类型均可能改变流程、字段、资料、SLA、派单和价格。若工单运行时读取“项目当前配置”，历史和在途结果将不可重现。

## 决策

所有可执行配置采用草稿—校验—审批—发布生命周期。发布产生不可变版本；工单创建时解析并锁定一个 `ConfigurationBundle`，记录所有适用版本引用及摘要。

## 后果

- 正面：历史可追溯、回放和重算，配置发布风险可控；
- 成本：需要版本依赖校验、引用保护、迁移工具和配置对比；
- 限制：在途工单默认不自动升级，升级必须是显式、受审计的迁移操作。
