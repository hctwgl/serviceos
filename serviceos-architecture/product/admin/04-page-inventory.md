# Admin 目标页面清单

本清单描述目标产品页面，不等同于当前路由清单。实施阶段必须把现有 Router、Page Registry、服务端 Navigation 和深链页面逐一映射到本清单，并给出保留、合并、重命名、迁移或删除入口的结论。

所有正式页面遵守 [经典专业风视觉与页面基线](12-classic-professional-visual-baseline.md)。

## 1. 工作台

| 页面 | 类型 | 核心任务 | 主要操作 |
|---|---|---|---|
| 运营工作台 | Workbench | 查看全局待办、SLA、异常和风险 | 下钻处理 |
| 我的待办 | List | 处理分配给本人或候选人的任务 | 领取、打开任务 |
| SLA 风险 | List/Workbench | 查看即将超时、已超时和暂停对象 | 催办、转交、打开对象 |
| 运营异常 | List | 查看自动化失败和业务阻塞 | 处理、转交、恢复 |

## 2. 工单运营

| 页面 | 类型 | 核心任务 | 主要操作 |
|---|---|---|---|
| 工单中心 | List | 查询全部授权范围内工单 | 查看、新建（有能力时）、批量导出（有能力时） |
| 工单详情 | Detail/Workspace | 在单一工作区理解 Workflow、Task、责任、资料、SLA、审核和外部回传 | allowed-actions 主动作 |
| 工单时间线 | Detail | 追溯业务事件 | 查看来源对象 |
| 待初审/待分配/待预约/上门中/待审核/整改中/已完成 | Saved View | 快速进入特定业务队列 | 与工单中心一致 |

## 3. 服务履约

| 页面 | 类型 | 核心任务 | 主要操作 |
|---|---|---|---|
| 服务任务 | List | 查询人工和自动任务 | 打开、领取、转交 |
| 任务详情 | Detail | 查看任务目标、输入、输出、责任和阻塞 | 执行 allowed-action |
| 预约管理 | List | 跟踪联系、预约、改约和失联 | 确认预约、改约、记录联系 |
| 网点派单 | List/Dedicated Flow | 将工单分配给服务网点 | 选择网点、确认分配 |
| 师傅调度 | Workbench/Dedicated Flow | 查看师傅负载并完成分配/改派 | 分配师傅、改派 |
| 勘测管理 | Saved View | 管理勘测阶段工单和任务 | 打开工单/任务 |
| 安装管理 | Saved View | 管理安装阶段工单和任务 | 打开工单/任务 |

## 4. 审核与整改

| 页面 | 类型 | 核心任务 | 主要操作 |
|---|---|---|---|
| 审核队列 | List | 查询待审核、本人审核和已处理记录 | 打开审核 |
| 勘测审核 | Saved View | 处理勘测资料审核 | 打开审核 |
| 安装审核 | Saved View | 处理安装资料审核 | 打开审核 |
| 平台终审 | Saved View | 处理平台最终审核 | 打开终审工作区 |
| 审核工作区 | Dedicated Flow | 对照资料、目标和规则作出决定 | 通过、退回补充、驳回整改 |
| 整改跟踪 | List | 跟踪整改责任、截止和复审 | 打开整改 |
| 整改详情/复审 | Detail/Dedicated Flow | 查看问题、整改证据和处理结果 | 提交整改、复审 |
| 审核记录 | List | 查询历史审核和决定 | 查看详情 |

## 5. 客户与项目

| 页面 | 类型 | 核心任务 | 主要操作 |
|---|---|---|---|
| 车企管理 | List | 管理车企基础资料和合作状态 | 新建、编辑、启停 |
| 车企详情 | Detail | 查看项目、接口和联系人 | 编辑、查看项目 |
| 项目管理 | List | 管理车企项目 | 新建项目、查看详情 |
| 项目新建/编辑 | Form | 维护项目基本信息 | 保存 |
| 项目详情 | Detail | 查看项目范围、网点、履约方案和版本 | 编辑、进入履约方案 |
| 项目履约方案管理 | List/Workbench | 在当前项目内查看多个履约方案及各方案的生效/草稿/历史版本、影响和发布记录 | 新建方案、编辑草稿、版本对比、发布、运行快照 |
| 履约方案配置 | List/Configuration | 查看并配置某个履约方案使用的 Workflow、Task、表单、资料、SLA 和策略 | 新增、编辑、预览 |
| 服务覆盖总览 | Workbench/Map | 查看项目在全国省市的网点覆盖、缺口和风险 | 下钻省市、打开网点或配置 |
| 全国服务覆盖分析 | Workbench/Analysis | 分析覆盖率、重叠覆盖、风险网点和城市明细 | 筛选、导出、进入区域配置 |
| 合作网点绑定 | Configuration | 维护项目可用网点 | 保存调整 |
| 接口与字段映射 | Configuration | 管理车企收单/回传映射 | 编辑草稿、校验、发布 |
| 配置发布记录 | List/Detail | 查看项目相关配置版本 | 查看版本、比较、影响分析 |

## 6. 组织与资源

| 页面 | 类型 | 核心任务 | 主要操作 |
|---|---|---|---|
| 企业组织 | Tree + List | 管理内部业务组织资料 | 新建、编辑、启停 |
| 合作组织 | List | 管理外部合作公司 | 新建、编辑、启停 |
| 合作组织详情 | Detail | 查看网点、联系人和合作状态 | 编辑、添加网点 |
| 服务网点 | List | 管理服务网点和能力 | 新建、编辑、停派 |
| 网点详情 | Detail | 查看区域、师傅、资质、容量和表现 | 编辑、管理师傅 |
| 师傅档案 | List | 查询师傅、所属网点、技能和状态 | 新建、编辑、停派 |
| 师傅详情 | Detail | 查看资质、能力、任务和历史 | 编辑、管理资质 |
| 技能与资质 | List/Configuration | 维护技能和资质目录 | 新建、编辑、停用 |
| 网点服务区域配置 | Tree + Map + Configuration | 按项目和业务类型配置网点负责的高德省市行政区域 | 保存、影响分析、确认生效 |
| 服务区域变更审阅 | Review/Impact | 对照新增、保留、移除和重叠区域并确认影响 | 保存草稿、提交确认、确认生效 |

## 7. 系统管理

| 页面 | 类型 | 核心任务 | 主要操作 |
|---|---|---|---|
| 用户管理 | List | 管理账号、组织、状态和角色摘要 | 新建、编辑、启停、重置安全设置 |
| 用户详情 | Detail | 查看账号、组织、角色、授权和登录记录 | 编辑、分配角色 |
| 组织管理 | Tree + List | 管理用户中心组织树和成员归属 | 新建、移动、编辑、启停 |
| 角色管理 | List | 管理角色模板 | 新建、复制、启停 |
| 角色详情 | Detail/Configuration | 管理菜单、操作权限、数据范围和成员 | 保存配置 |
| 菜单管理 | Tree/Configuration | 管理 Page Registry/Navigation 的产品化配置 | 编辑名称、分组、顺序、启用条件 |
| 权限管理 | List/Detail | 查看 Capability 的业务名称、说明和使用关系 | 查看、维护展示元数据 |
| 数据权限 | Configuration | 配置租户、项目、区域、组织和网点范围 | 保存范围 |
| 授权与委托 | List/Form | 管理临时或长期授权 | 新建、撤销 |
| 登录与安全 | List/Detail | 查看登录策略、会话和安全事件 | 结束会话、调整策略（有能力时） |

## 8. 审计与监控

| 页面 | 类型 | 核心任务 | 主要操作 |
|---|---|---|---|
| 操作日志 | List/Detail | 查询业务和管理操作 | 查看详情、导出（受控） |
| 登录日志 | List/Detail | 查询登录成功、失败和风险 | 查看详情 |
| 配置变更 | List/Detail | 追溯草稿、校验、发布和生效 | 查看版本、比较 |
| 接口记录 | List/Detail | 查询收单、回传和通知交付 | 查看 attempt、重试（有能力时） |
| 任务失败 | List/Detail | 查询自动任务失败和人工接管 | 重试、接管（有能力时） |
| 系统诊断 | Diagnostic | 查看 correlationId、版本、checkpoint 和内部状态 | 复制诊断信息 |

## 10. 页面记录要求

实施时，每个页面必须补充以下产品元数据：

- pageId；
- 路由；
- 所属菜单；
- 目标角色；
- 核心任务；
- 页面类型；
- 主操作；
- capability；
- scope；
- allowed-actions；
- 后端 API；
- 正常/空/错/无权限/冲突状态；
- 产品状态；
- 视觉验收状态；
- 适用的经典专业风页面模式；
- 对应批准视觉参考图及并排对照证据；
- 1440×1024 和 1280px 截图证据。

未完成页面不得仅凭“路由可访问”或“样式接近概念图”标记为已完成。
