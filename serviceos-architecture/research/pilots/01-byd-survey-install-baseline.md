# 比亚迪勘安试点业务基线

## 1. 目的

冻结 ServiceOS 首个端到端试点范围，避免在业务事实尚未稳定时同时开发全部车企、品牌、区域和业务类型。本文件只有在业务、产品和技术负责人共同确认后才能升级为 Accepted。

## 2. 已确认范围

| 维度 | 基线 |
|---|---|
| 车企集团 | 比亚迪 |
| 品牌 | 海洋 |
| 业务类型 | 勘测后安装（勘安） |
| 区域 | 山东省 |
| 工单来源 | 车企接口；接口未就绪时允许受控导入模拟 |
| 总部角色 | 品牌负责人、客服经理、项目经理、跟进专员、客服/审核员 |
| 履约角色 | 网点负责人、师傅 |
| 结算 | 首期影子计价，不作为正式财务权威 |
| 外部审核 | 总部审核后回传车企，车企驳回先进入客服协调任务 |

## 3. 首期主流程

```text
车企推送工单
→ 自动分配品牌负责人、客服经理、项目经理
→ 可配置客服初审
→ 自动派网点，失败转项目经理
→ 系统或网点分配师傅
→ 勘测预约
→ 上门勘测
→ 提交勘测资料
→ 客服审核
→ 安装预约
→ 上门安装
→ 提交安装资料
→ 客服审核
→ 自动回传车企，失败转客服
→ 生成履约事实
→ 对上/对下影子计价
→ 完成
```

## 4. 首期必须覆盖的异常

- 重复工单推送；
- 配置未命中；
- 地址无法匹配山东省内区域；
- 无可派网点；
- 网点达到在途上限；
- 自动分配师傅失败；
- 用户改约或无法联系；
- 勘测不通过；
- 勘测/安装资料单项驳回并多轮补传；
- 用户取消；
- 回传车企失败、重复回执和车企再次驳回；
- 强制关闭和工单恢复。

## 5. 当前已确认规则

### 5.1 分配与派单

- 跟进责任人由系统按品牌、项目和区域规则分配；
- 试点品牌限定为比亚迪海洋；
- 试点区域限定为山东省；
- 自动派网点优先，异常由项目经理处理；
- 同一区域允许多个可派网点；
- 黑名单、停派、品牌/区域、业务能力、资质和容量属于硬过滤；
- 当前容量按在途工单量控制，按业务类型分别设置；
- 月度签约比例用于候选网点倾斜，但履约质量可以覆盖比例要求；
- 无候选网点时不自动跨区，转人工；
- 人工改派必须填写原因，原网点失去工单查看权限。

### 5.2 资料与审核

- 比亚迪安装资料最多约 30 项；
- 大部分固定，部分根据施工条件动态出现；
- 支持现场拍摄、GPS、水印、OCR、重复图和质量检测等项目级配置；
- OCR 需要识别并核对桩 SN、VIN 等内容；
- 单项资料可独立驳回并多轮补传；
- 审核员不能修改师傅字段，也不能代传资料；
- 已审核资料和历史版本不可覆盖；
- 特殊人员可强制通过，但必须高风险授权和审计。

### 5.3 SLA

- 各节点时效按车企、业务类型和风险等级配置；
- 夜间、周末、节假日、用户原因、物业、电力报装和缺货等暂停规则按项目配置；
- 暂停当前不要求审批，但必须记录原因、时间和操作者；
- 超时影响网点考核并升级给品牌经理；
- 自动派单失败当前基线为 24 小时内处理，并需要提前预警。

## 6. 必须由业务负责人补齐

| 编号 | 待确认项 | 阻塞范围 |
|---:|---|---|
| 1 | 实际车企接口和报文样本 | 接入适配器和幂等键 |
| 2 | 初审是否启用 | 工作流路由 |
| 3 | 师傅分配模式 | 派单与网点端产品 |
| 4 | 勘测不通过的最终处置 | 关闭、暂停或转其他业务 |
| 5 | 每个节点实际 SLA | 时钟、预警和升级 |
| 6 | 勘测和安装字段清单 | Form Schema 与物理模型 |
| 7 | 资料项、条件和驳回原因 | Evidence Schema 与审核 |
| 8 | 对上/对下价格规则样本 | 影子计价和差异对账 |
| 9 | 车企回传字段和审核回执 | Integration Contract |
| 10 | 强制关闭和恢复审批人 | 高风险权限与补偿 |
| 11 | 山东省内区域、网点、负责人和容量清单 | Dispatch Policy 与组织数据 |
| 12 | 海洋品牌适用桩型、车型和资料差异 | 配置作用域与条件规则 |

## 7. 样本 Gate

进入业务编码前必须提供脱敏数据：

- 至少 20 个正常勘安工单；
- 至少 20 个异常工单，覆盖第 4 节场景；
- 工单状态和操作日志；
- 勘测及安装资料样例；
- 总部和车企审核记录；
- 对上、对下结算结果或人工计算表；
- 至少一个规则变更前后的历史案例。

## 8. 试点成功标准

- 新工单唯一命中并锁定 Configuration Bundle；
- 主流程和已列异常可通过命令、事件和任务恢复执行；
- 配置发布不改变运行中工单；
- 资料、审核、派单和回传结果全链路可解释；
- 重复请求、消息和离线同步不造成重复推进；
- 影子计价与人工结果的差异可定位到事实或规则；
- 所有高风险操作有授权、原因和审计记录。

## 9. 非范围

首期不承诺：

- 比亚迪王朝、腾势、方程豹、仰望等其他品牌；
- 山东省以外区域；
- 维修、拆装和其他业务；
- 正式财务入账；
- 完整 BPMN/低代码设计器；
- AI 自动审核直接替代人工；
- SaaS 商业化和全行业通用能力。
